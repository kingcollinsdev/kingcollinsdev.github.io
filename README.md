# Portfolio Website

The project is a portfolio website where recruiters or anyone can view the projects i have created and also learn more about me.

## Features

- Feature 1: You can view my revelant skills. 
- Feature 2: You can view my resume.
- Feature 3: All of the projects are displayed.
- Responsive design
- Cross-browser compatible

## Technologies Used

- **HTML5** - Structure and content
- **CSS3** - Styling and layout


## Installation

1. Clone the repository:
```bash
gh repo clone kingcollinsdev/kingcollinsdev.github.io
```

2. Navigate to the project directory:
```bash
cd kingcollinsdev.github.io
```
